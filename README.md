# MiniProyecto_GestorDeTareas

## Objetivo del Sistema

La gestión del tiempo y la organización de las actividades son factores fundamentales para mejorar la productividad de las personas, especialmente en el ámbito académico y personal. Los usuarios suelen manejar múltiples responsabilidades diariamente, como tareas académicas, actividades personales, recordatorios importantes o compromisos laborales.

En muchos casos, estas actividades se gestionan de manera informal, por ejemplo mediante notas físicas, recordatorios mentales o aplicaciones digitales poco estructuradas. Esto puede provocar desorganización, olvidos o dificultades para priorizar tareas importantes.

Ante esta situación, surge la necesidad de desarrollar herramientas digitales que permitan a los usuarios organizar, registrar, visualizar, modificar y dar seguimiento al estado de sus tareas de forma eficiente, facilitando un mejor control de sus pendientes.

El presente proyecto consiste en el diseño de una **aplicación web de gestión de tareas personales**, que permite a los usuarios administrar sus actividades de manera sencilla y organizada. Entre las funcionalidades principales del sistema se encuentran:

- Creación de tareas
- Visualización de listas de tareas
- Edición de tareas existentes
- Eliminación de tareas
- Actualización del estado de las actividades

El sistema está construido utilizando una **arquitectura moderna**, lo que permite mantener una estructura organizada y escalable. La interfaz facilita la interacción del usuario con el sistema, valida la información ingresada y gestiona la comunicación con la base de datos para garantizar el almacenamiento permanente de la información.
